## 📖 Tentang Project

**Robloxin Aja** adalah website fan-made yang dibuat untuk para penggemar Roblox, khususnya komunitas Indonesia. Website ini menyediakan informasi lengkap seputar dunia Roblox — mulai dari sejarah platform, rekomendasi game populer, inspirasi tampilan avatar, hingga galeri foto saat bermain.

Website ini dibuat sebagai proyek pembelajaran pengembangan web statis dengan tampilan yang menarik dan navigasi yang mudah digunakan.

---

## ✨ Fitur

- 🏠 **Beranda** — Halaman sambutan dengan hero image Roblox dan pengenalan website
- 📚 **Tentang** — Informasi lengkap tentang Roblox, sejarah, cara kerja, dan komunitas
- 🎮 **Rekomendasi Game** — Daftar game Roblox populer beserta deskripsinya
- 👤 **Avatar & Item** — Inspirasi tampilan avatar keren dan koleksi item menarik
- 🖼️ **Galeri** — Kumpulan foto/screenshot ketika bermain Roblox
- 📬 **Formulir Kontak** — Form untuk menghubungi pemilik website
- 📱 **Responsif** — Tampilan menyesuaikan di semua ukuran layar

---

## 📄 Struktur Halaman

| Halaman | Deskripsi |
|---|---|
| `Beranda` | Halaman utama dengan hero image dan sambutan |

## 🚀 Cara Menjalankan

### Prasyarat
Tidak diperlukan instalasi khusus. Cukup browser modern (Chrome, Firefox, Edge, dll).

### Langkah-langkah

1. **Clone atau unduh repository ini:**
   ```bash
   git clone https://github.com/username/robloxin-aja.git
   ```

2. **Masuk ke folder project:**
   ```bash
   cd robloxin-aja
   ```

3. **Buka file `index.html` di browser:**
   ```bash
   # Klik dua kali file index.html
   # atau gunakan Live Server di VS Code
   ```

> 💡 **Rekomendasi:** Gunakan ekstensi **Live Server** di Visual Studio Code untuk pengalaman development yang lebih baik.
